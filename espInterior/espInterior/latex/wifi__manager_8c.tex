\doxysubsection{Descripción detallada}
Implementación del gestor de conectividad Wi\+Fi. 

Este módulo implementa la inicialización, conexión y monitoreo de la interfaz Wi\+Fi del ESP32 utilizando el framework ESP-\/\+IDF.

Funcionalidades principales:\+
\begin{DoxyItemize}
\item Inicialización del subsistema Wi\+Fi.
\item Gestión automática de conexión y reconexión.
\item Obtención de dirección IP mediante DHCP.
\item Escaneo de redes inalámbricas disponibles.
\item Consulta del estado de conexión.
\item Sincronización mediante Event Groups de Free\+RTOS.
\end{DoxyItemize}

El módulo abstrae la complejidad de la pila de red y proporciona una interfaz sencilla para las tareas de aplicación.

Mecanismos Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Event Groups para sincronización de estados de conexión.
\end{DoxyItemize}

Eventos gestionados:\+
\begin{DoxyItemize}
\item Inicio del modo estación.
\item Conexión y desconexión Wi\+Fi.
\item Obtención de dirección IP.
\item Reintentos automáticos de conexión.
\end{DoxyItemize}
