\doxysubsection{Descripción detallada}
Implementación del módulo de almacenamiento en tarjeta SD mediante SPI. 

Este módulo proporciona funciones para:\+
\begin{DoxyItemize}
\item Inicialización y montaje de la tarjeta SD.
\item Escritura de registros históricos en formato CSV.
\item Almacenamiento temporal de mensajes MQTT pendientes.
\item Recuperación de mensajes pendientes.
\item Gestión segura de acceso concurrente mediante mutex.
\end{DoxyItemize}

El módulo permite mantener la persistencia de datos del sistema de fermentación incluso ante fallos de conectividad MQTT.

El acceso al bus SPI se encuentra protegido mediante exclusión mutua para evitar condiciones de carrera entre tareas Free\+RTOS.
