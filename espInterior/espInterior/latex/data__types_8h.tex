\doxysubsection{Descripción detallada}
Definición de estructuras de datos compartidas del sistema. 

Este archivo contiene las estructuras utilizadas para el intercambio de información entre las tareas de Free\+RTOS mediante colas de comunicación.

Define los tipos de datos empleados para:\+
\begin{DoxyItemize}
\item Gestión de fecha y hora.
\item Adquisición de sensores.
\item Comandos de control.
\item Información para visualización.
\end{DoxyItemize}

Estas estructuras permiten desacoplar las tareas del sistema, facilitando una arquitectura modular y escalable.
