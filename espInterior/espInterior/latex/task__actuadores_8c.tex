\doxysubsection{Descripción detallada}
Implementación de la tarea de ejecución de actuadores. 

Esta tarea recibe comandos generados por la lógica de control mediante una cola Free\+RTOS y ejecuta las acciones físicas correspondientes sobre los actuadores del sistema.

Actuadores gestionados:\+
\begin{DoxyItemize}
\item Bomba de circulación.
\item Motor de mezcla.
\item Servomotor.
\end{DoxyItemize}

Características:\+
\begin{DoxyItemize}
\item Consumo de comandos mediante colas Free\+RTOS.
\item Control temporizado del motor de mezcla.
\item Notificación a la tarea de energía al finalizar una mezcla.
\item Separación entre lógica de control y acceso a hardware.
\end{DoxyItemize}

Mecanismos Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Colas.
\item Notificaciones entre tareas.
\end{DoxyItemize}

Tarea encargada de ejecutar acciones físicas. 

Tarea de ejecución de actuadores.

Esta tarea permanece bloqueada esperando comandos provenientes de queue\+\_\+control.

Funciones principales:\+
\begin{DoxyItemize}
\item Activación y desactivación de la bomba.
\item Posicionamiento del servomotor.
\item Gestión del motor de mezcla.
\item Notificación a task\+\_\+energia al finalizar una mezcla.
\end{DoxyItemize}

La tarea no implementa lógica de decisión; únicamente ejecuta los comandos generados por task\+\_\+control.

Tarea encargada de ejecutar acciones físicas.

Espera comandos provenientes de la cola de control y ejecuta las acciones correspondientes sobre los actuadores físicos del sistema.

La tarea opera como consumidora de estructuras \doxylink{structcontrol__cmd__t}{control\+\_\+cmd\+\_\+t}, permitiendo separar la lógica de decisión de la manipulación directa del hardware.
