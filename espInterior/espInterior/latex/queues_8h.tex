\doxysubsection{Descripción detallada}
Declaración de las colas de comunicación del sistema. 

Este módulo define las colas Free\+RTOS utilizadas para el intercambio de información entre las diferentes tareas del sistema embebido.

Las colas permiten una comunicación segura y desacoplada entre productores y consumidores, evitando el uso de variables globales compartidas.

Flujo principal:\+

task\+\_\+sensores \texorpdfstring{$\vert$}{|} v queue\+\_\+sensores \texorpdfstring{$\vert$}{|} v task\+\_\+control \texorpdfstring{$\vert$}{|} v queue\+\_\+control \texorpdfstring{$\vert$}{|} v task\+\_\+actuadores

Además, los datos pueden ser distribuidos hacia:\+
\begin{DoxyItemize}
\item queue\+\_\+logger
\item queue\+\_\+display
\item queue\+\_\+mqtt
\end{DoxyItemize}
