\doxysubsection{Descripción detallada}
Implementación de la tarea de registro de datos. 

Esta tarea recibe mediciones provenientes de los sensores, genera registros en formato CSV y los almacena de forma persistente en la tarjeta SD.

Funciones principales:\+
\begin{DoxyItemize}
\item Recepción de datos mediante colas Free\+RTOS.
\item Formateo de registros CSV.
\item Escritura en memoria SD.
\item Generación de historial de fermentación.
\end{DoxyItemize}

Arquitectura:\+

queue\+\_\+logger \texorpdfstring{$\vert$}{|} v task\+\_\+logger \texorpdfstring{$\vert$}{|} v SD Card

La tarea se ejecuta de forma independiente para evitar que las operaciones de almacenamiento bloqueen la adquisición de sensores o la lógica de control.

Mecanismos Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Colas para comunicación entre tareas.
\end{DoxyItemize}

Tarea principal de registro de datos. 

Espera mediciones provenientes de queue\+\_\+logger, genera una línea en formato CSV y la almacena en la tarjeta SD.

Cada registro contiene:\+
\begin{DoxyItemize}
\item Fecha.
\item Hora.
\item Temperatura.
\item pH.
\item CO2 procesado.
\item CO2 crudo.
\item Voltaje.
\item Corriente.
\end{DoxyItemize}

La tarea permanece bloqueada mientras no existan nuevos datos disponibles, optimizando el uso de CPU.
