\doxysubsection{Descripción detallada}
Declaración de la tarea de registro de datos. 

Este módulo define la tarea responsable de almacenar las mediciones generadas durante el proceso de fermentación.

La tarea recibe datos provenientes de los sensores mediante una cola Free\+RTOS, los formatea adecuadamente y los registra en la tarjeta SD para su posterior análisis.

Funciones principales:\+
\begin{DoxyItemize}
\item Recepción de mediciones del sistema.
\item Formateo de registros en formato CSV.
\item Almacenamiento persistente en tarjeta SD.
\item Conservación del historial de fermentación.
\end{DoxyItemize}

Arquitectura:\+

queue\+\_\+logger \texorpdfstring{$\vert$}{|} v task\+\_\+logger \texorpdfstring{$\vert$}{|} v SD Card

Esta tarea permite desacoplar el almacenamiento de datos de las tareas de adquisición y control.

Tarea principal de registro de datos. 

Espera información proveniente de queue\+\_\+logger, genera registros en formato CSV y los almacena en la tarjeta SD.

La tarea se ejecuta de forma independiente para evitar que las operaciones de escritura afecten el tiempo de respuesta de las tareas críticas del sistema.


\begin{DoxyParams}{Parámetros}
{\em pv\+Parameters} & Parámetros de la tarea (no utilizados).\\
\hline
\end{DoxyParams}
Espera mediciones provenientes de queue\+\_\+logger, genera una línea en formato CSV y la almacena en la tarjeta SD.

Cada registro contiene:\+
\begin{DoxyItemize}
\item Fecha.
\item Hora.
\item Temperatura.
\item pH.
\item CO2 procesado.
\item CO2 crudo.
\item Voltaje.
\item Corriente.
\end{DoxyItemize}

La tarea permanece bloqueada mientras no existan nuevos datos disponibles, optimizando el uso de CPU.
