\doxysubsection{Descripción detallada}
Implementación de la capa de abstracción para el ADC del ESP32. 

Este módulo implementa el acceso al convertidor analógico-\/digital utilizando el modo One-\/\+Shot proporcionado por ESP-\/\+IDF.

Funcionalidades:\+
\begin{DoxyItemize}
\item Inicialización del ADC1.
\item Configuración de canales analógicos.
\item Calibración mediante Line Fitting.
\item Conversión de valores ADC a voltaje.
\end{DoxyItemize}

Los sensores analógicos del sistema utilizan este módulo como interfaz única de acceso al ADC.

Sensores asociados:\+
\begin{DoxyItemize}
\item MQ135
\item Sensor de pH
\item Sensor de voltaje
\end{DoxyItemize}
