\doxysubsection{Descripción detallada}
Implementación de la gestión energética del sistema. 

Esta tarea coordina el ciclo operativo general del sistema de fermentación y gestiona la transición hacia modos de bajo consumo.

El flujo de operación sigue la secuencia:\+

Medir ↓ Procesar ↓ Actuar ↓ Registrar ↓ Dormir

La sincronización entre tareas se realiza mediante notificaciones de Free\+RTOS, permitiendo una ejecución eficiente y evitando ciclos de espera activa.

Funciones principales:\+
\begin{DoxyItemize}
\item Inicio de ciclos de adquisición.
\item Coordinación entre tareas.
\item Espera de finalización de procesos.
\item Activación del modo Deep Sleep.
\end{DoxyItemize}

Mecanismos Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Notificaciones entre tareas.
\end{DoxyItemize}

Modos de bajo consumo:\+
\begin{DoxyItemize}
\item Deep Sleep.
\end{DoxyItemize}

Tarea principal de gestión energética. 

La tarea permanece bloqueada esperando una notificación que indique el inicio de un nuevo ciclo operativo.

Flujo de ejecución:\+
\begin{DoxyItemize}
\item Esperar activación.
\item Iniciar adquisición de sensores.
\item Esperar finalización de procesos.
\item Entrar en modo Deep Sleep.
\end{DoxyItemize}

La coordinación se realiza mediante notificaciones directas de Free\+RTOS para minimizar el uso de memoria y CPU.


\begin{DoxyParams}{Parámetros}

\hline
\end{DoxyParams}
Coordina el ciclo de ejecución del sistema y supervisa la finalización de las tareas necesarias antes de activar mecanismos de ahorro energético.

Dependiendo del estado del sistema, puede preparar la transición hacia modos de bajo consumo como Deep Sleep.

El sistema permanecerá suspendido hasta que ocurra el evento de despertar configurado.
