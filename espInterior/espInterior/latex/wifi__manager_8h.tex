\doxysubsection{Descripción detallada}
Interfaz para la gestión de conectividad Wi\+Fi. 

Este módulo proporciona las funciones necesarias para inicializar la interfaz Wi\+Fi del ESP32, establecer conexión con una red inalámbrica y consultar el estado de conectividad.

Funcionalidades:\+
\begin{DoxyItemize}
\item Inicialización del subsistema Wi\+Fi.
\item Conexión automática a una red configurada.
\item Espera sincronizada de conexión.
\item Escaneo de redes disponibles.
\item Consulta del estado de conexión.
\end{DoxyItemize}

El módulo abstrae la complejidad de la configuración Wi\+Fi del ESP-\/\+IDF y ofrece una interfaz simplificada para las tareas de aplicación.

Fernando Plazas Trujillo Isabella Ordoñez Juan Daniel Constain
