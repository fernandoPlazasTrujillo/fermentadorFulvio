\doxysubsection{Descripción detallada}
Implementación del gestor de comunicación MQTT. 

Este módulo implementa la conexión y comunicación con un broker MQTT utilizando la biblioteca ESP-\/\+MQTT del ESP-\/\+IDF.

Funcionalidades principales:\+
\begin{DoxyItemize}
\item Inicialización del cliente MQTT.
\item Gestión de eventos de conexión y desconexión.
\item Publicación de mensajes en tópicos MQTT.
\item Sincronización mediante Event Groups de Free\+RTOS.
\item Control del estado de conexión.
\end{DoxyItemize}

El módulo abstrae la complejidad del protocolo MQTT y proporciona una interfaz sencilla para las tareas de aplicación.

Mecanismos Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Event Groups para sincronización de eventos MQTT.
\end{DoxyItemize}

Eventos gestionados:\+
\begin{DoxyItemize}
\item Conexión al broker.
\item Desconexión del broker.
\item Confirmación de publicación.
\item Errores de comunicación.
\end{DoxyItemize}
