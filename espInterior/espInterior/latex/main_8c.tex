\doxysubsection{Descripción detallada}
Punto de entrada del sistema de fermentación basado en ESP32. 

Este módulo realiza la inicialización completa del sistema, configura los periféricos necesarios, crea las tareas Free\+RTOS y pone en marcha el ciclo operativo de adquisición, control, almacenamiento y comunicación.

Funciones principales:\+
\begin{DoxyItemize}
\item Inicialización de hardware.
\item Inicialización de red Wi\+Fi.
\item Configuración del RTC DS3231.
\item Configuración de wakeup para Deep Sleep.
\item Creación de colas Free\+RTOS.
\item Creación de tareas del sistema.
\item Inicio del ciclo operativo.
\end{DoxyItemize}
