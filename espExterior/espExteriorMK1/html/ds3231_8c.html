Arquitectura del sistema

El módulo DS3231 constituye la referencia temporal principal del sistema distribuido de fermentación de café.

Sus funciones principales son:

  • Generación de timestamps para almacenamiento en SD.
  • Sincronización de mediciones ambientales.
  • Activación periódica del sistema mediante alarmas.
  • Soporte para estrategias de ahorro energético basadas en sleep.

◆ ds3231_clear_alarm_flag()

int ds3231_clear_alarm_flag ( void )

Limpia el indicador de alarma A1F.

Limpia el indicador interno de alarma.

Después de que una alarma genera una interrupción, el bit A1F permanece activado dentro del registro de estado del DS3231.

Esta función limpia dicho bit para permitir que la siguiente alarma vuelva a producir una interrupción válida.

Registro utilizado:

  • Status Register (0x0F)
Valores devueltos
0Operación completada.
Atención
Si esta función no se ejecuta después de atender la interrupción, las alarmas posteriores no funcionarán correctamente.

◆ ds3231_set_alarm_every_minute()

int ds3231_set_alarm_every_minute ( void )

Configura una alarma periódica cada minuto.

Programa la alarma 1 del DS3231 para generar una interrupción cuando los segundos alcancen el valor 0.

La configuración utilizada permite generar una interrupción exactamente al inicio de cada minuto.

Registros utilizados:

  • Alarm1 (0x07 - 0x0A)
  • Control (0x0E)

Esta funcionalidad se utiliza para despertar periódicamente al ESP32 desde modos de bajo consumo.

Valores devueltos
0Configuración exitosa.
-1Error de comunicación I2C.
Nota
El pin INT/SQW del DS3231 debe estar conectado a un GPIO configurado como interrupción externa.

Utiliza la alarma 1 del DS3231 para generar una interrupción exactamente cuando los segundos alcanzan el valor cero.

Esta funcionalidad es utilizada para despertar el sistema desde modos de bajo consumo y ejecutar los ciclos de medición.

Valores devueltos
0Configuración exitosa.
-1Error de comunicación I2C.
Nota
Requiere que el pin INT/SQW del DS3231 esté conectado a un GPIO configurado como interrupción externa.
