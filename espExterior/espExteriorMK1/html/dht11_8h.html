Descripción detallada

Interfaz pública del controlador para el sensor DHT11.

Este módulo proporciona las definiciones y funciones necesarias para obtener mediciones de temperatura y humedad relativa utilizando el sensor DHT11.

El sensor utiliza un protocolo propietario de comunicación basado en temporización precisa de pulsos digitales.

Nota
La implementación del protocolo se encuentra en dht11.c.

int dht11_read ( dht11_data_t * data)

Realiza una lectura completa del sensor DHT11.

Esta función ejecuta el protocolo de comunicación requerido por el sensor, recibe los datos transmitidos y verifica su integridad mediante checksum.

Si la lectura es exitosa, los valores obtenidos son almacenados en la estructura indicada por el parámetro data.

Parámetros
[out]dataPuntero a la estructura donde se almacenarán los datos leídos de temperatura y humedad.
Valores devueltos
0Lectura realizada correctamente.
-1Error de comunicación o ausencia de respuesta del sensor.
-2Error de verificación de checksum.
Atención
Esta función es bloqueante debido a los requerimientos de temporización del protocolo DHT11.
Nota
Se recomienda proteger el acceso mediante mutex si el sensor puede ser utilizado desde múltiples tareas FreeRTOS.

Ejecuta la secuencia completa de comunicación:

  1. Envía la señal de inicio al sensor.
  2. Espera la respuesta de presencia.
  3. Lee los 40 bits de información transmitidos.
  4. Verifica la integridad mediante checksum.
  5. Extrae los valores de temperatura y humedad.

Los datos obtenidos se almacenan en la estructura proporcionada por el usuario.

Parámetros
[out]dataEstructura donde se almacenarán los valores leídos.
Valores devueltos
0Lectura realizada correctamente.
-1No se recibió respuesta del sensor.
-2Error de checksum en los datos recibidos.
Nota
Esta función es bloqueante debido al uso de retardos de microsegundos requeridos por el protocolo DHT11.
