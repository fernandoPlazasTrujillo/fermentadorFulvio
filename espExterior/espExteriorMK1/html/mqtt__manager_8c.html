Descripción detallada

Implementación del gestor de comunicación MQTT.

Este módulo implementa la conexión con un broker MQTT utilizando la biblioteca ESP-MQTT de ESP-IDF.

La sincronización entre eventos MQTT y tareas de FreeRTOS se realiza mediante Event Groups, permitiendo notificar:

  • Estado de conexión.
  • Confirmación de publicaciones.
  • Errores de comunicación.

El módulo proporciona una interfaz desacoplada para que otras tareas puedan publicar información sin interactuar directamente con el cliente MQTT.

Manejador de eventos MQTT.

Recibe los eventos generados por la biblioteca ESP-MQTT y actualiza el grupo de eventos utilizado por FreeRTOS para sincronizar las tareas de aplicación.

Eventos gestionados:

  • MQTT_EVENT_CONNECTED
  • MQTT_EVENT_DISCONNECTED
  • MQTT_EVENT_PUBLISHED
  • MQTT_EVENT_ERROR
