\doxysubsection{Descripción detallada}
Gestión de conectividad Wi\+Fi para el ESP32. 

Este módulo proporciona una interfaz para la inicialización, conexión y monitoreo de la red Wi\+Fi utilizada por el sistema.

Responsabilidades:\+
\begin{DoxyItemize}
\item Inicializar la pila TCP/\+\+IP.
\item Configurar la interfaz Wi\+Fi.
\item Gestionar la conexión a la red inalámbrica.
\item Verificar el estado de conexión.
\item Realizar escaneo de redes disponibles.
\end{DoxyItemize}

Este módulo abstrae los detalles de la implementación Wi\+Fi, permitiendo que otras tareas del sistema utilicen servicios de red sin depender directamente de ESP-\/\+IDF.
