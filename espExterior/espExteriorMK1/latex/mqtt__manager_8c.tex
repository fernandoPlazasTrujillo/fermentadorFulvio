\doxysubsection{Descripción detallada}
Implementación del gestor de comunicación MQTT. 

Este módulo implementa la conexión con un broker MQTT utilizando la biblioteca ESP-\/\+MQTT de ESP-\/\+IDF.

La sincronización entre eventos MQTT y tareas de Free\+RTOS se realiza mediante Event Groups, permitiendo notificar:\+


\begin{DoxyItemize}
\item Estado de conexión.
\item Confirmación de publicaciones.
\item Errores de comunicación.
\end{DoxyItemize}

El módulo proporciona una interfaz desacoplada para que otras tareas puedan publicar información sin interactuar directamente con el cliente MQTT.
