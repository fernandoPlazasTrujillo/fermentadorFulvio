\doxysubsection{Descripción detallada}
Punto de entrada del sistema ESP32 para monitoreo ambiental. 

Este módulo inicializa el sistema, configura periféricos y coordina la ejecución de tareas bajo Free\+RTOS.

Funcionalidades principales:\+
\begin{DoxyItemize}
\item Inicialización del bus I2C
\item Inicialización de la tarjeta SD
\item Creación de colas de comunicación
\item Creación de tareas del sistema
\item Configuración del RTC DS3231 para wakeup periódico
\item Gestión de bajo consumo mediante Deep Sleep
\end{DoxyItemize}

Arquitectura:\+
\begin{DoxyItemize}
\item task\+\_\+sensores → adquisición de datos
\item task\+\_\+logger → almacenamiento en SD
\item task\+\_\+display → visualización en LCD
\item task\+\_\+mqtt → transmisión MQTT
\end{DoxyItemize}

Flujo general:\+
\begin{DoxyEnumerate}
\item Inicialización del sistema
\item Conexión Wi\+Fi
\item Creación de tareas
\item Espera finalización de tareas críticas (display y MQTT)
\item Configuración de alarma RTC
\item Entrada en Deep Sleep
\end{DoxyEnumerate}

Aspectos de Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Creación dinámica de tareas.
\item Colas para intercambio de datos.
\item Notificaciones de tareas para sincronización.
\item Bloqueo eficiente mediante port\+MAX\+\_\+\+DELAY.
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
El sistema funciona en ciclos:\+ medir → procesar → mostrar → guardar → dormir
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
El pin SQW del DS3231 debe estar correctamente conectado para permitir el wakeup desde Deep Sleep.
\end{DoxyWarning}
