\doxysubsection{Descripción detallada}
Tarea encargada de la visualización de datos en LCD. 

Esta tarea recibe datos del sistema a través de una cola y los muestra en una pantalla LCD 16x2.

Responsabilidades:\+
\begin{DoxyItemize}
\item Formatear datos para visualización
\item Actualizar la pantalla LCD
\item Sincronizar con la tarea principal mediante notificaciones
\end{DoxyItemize}

Flujo de ejecución:\+
\begin{DoxyEnumerate}
\item Espera datos desde la cola (bloqueante)
\item Formatea la información (hora, fecha, temperatura, humedad)
\item Actualiza el display
\item Notifica a la tarea principal
\end{DoxyEnumerate}

Comunicación:\+
\begin{DoxyItemize}
\item Entrada:\+ cola\+\_\+display (Queue)
\item Salida:\+ notificación a main\+\_\+task\+\_\+handle
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Esta tarea es bloqueante sobre la cola (port\+MAX\+\_\+\+DELAY), lo cual optimiza el consumo energético.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
El acceso al LCD no es thread-\/safe. Esta tarea debe ser la única que interactúe con el display o debe usarse un mutex.
\end{DoxyWarning}
