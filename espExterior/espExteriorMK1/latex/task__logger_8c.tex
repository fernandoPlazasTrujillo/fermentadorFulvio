\doxysubsection{Descripción detallada}
Tarea encargada del almacenamiento de datos en tarjeta SD. 

Esta tarea recibe datos ambientales desde una cola y los almacena en una tarjeta SD en formato CSV.

Responsabilidades:\+
\begin{DoxyItemize}
\item Recibir datos desde otras tareas (sensores/\+control)
\item Formatear la información en una línea estructurada
\item Guardar los datos en la SD
\item Mostrar información de depuración en consola
\end{DoxyItemize}

Flujo de ejecución:\+
\begin{DoxyEnumerate}
\item Espera datos en la cola (bloqueante)
\item Formatea los datos en formato CSV
\item Imprime en consola (debug)
\item Guarda en tarjeta SD
\end{DoxyEnumerate}

Comunicación:\+
\begin{DoxyItemize}
\item Entrada:\+ cola\+\_\+datos (Queue)
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Esta tarea es bloqueante sobre la cola, lo que optimiza el uso de CPU y energía.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
El acceso a la SD no es thread-\/safe. Se recomienda usar un mutex SPI si múltiples tareas acceden al bus.
\end{DoxyWarning}
