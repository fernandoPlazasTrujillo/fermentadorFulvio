\doxysubsection{Descripción detallada}
Tarea encargada de la transmisión MQTT de datos ambientales. 

Esta tarea recibe datos desde una cola Free\+RTOS, genera un mensaje JSON y lo publica en un broker MQTT.

Para aumentar la confiabilidad del sistema, los mensajes que no pueden ser enviados debido a fallos de conectividad son almacenados temporalmente en la tarjeta SD y reenviados automáticamente cuando la conexión se restablece.

Responsabilidades:\+
\begin{DoxyItemize}
\item Recibir datos desde la cola MQTT.
\item Construir el payload JSON.
\item Gestionar la conexión MQTT.
\item Publicar datos en el broker.
\item Almacenar mensajes pendientes en SD.
\item Reenviar mensajes pendientes.
\item Notificar la finalización del ciclo.
\end{DoxyItemize}

Aspectos de Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Queue para recepción de datos.
\item Task Notification para sincronización.
\item Eliminación dinámica de tarea mediante v\+Task\+Delete().
\end{DoxyItemize}

Estrategia de tolerancia a fallos:\+
\begin{DoxyItemize}
\item Si MQTT falla, los datos se almacenan en SD.
\item Al recuperar la conexión se reenvían los mensajes pendientes.
\item Se evita la pérdida de información durante cortes de red.
\end{DoxyItemize}

\begin{DoxyEnumerate}
\item Espera datos desde la cola MQTT.
\item Construye el mensaje JSON.
\item Inicia el cliente MQTT.
\item Espera conexión con el broker.

\item Finaliza su ejecución.

\begin{DoxyItemize}
