\doxysubsection{Descripción detallada}
Implementación del módulo de almacenamiento en tarjeta SD. 

Este módulo implementa la gestión de una tarjeta SD conectada mediante interfaz SPI utilizando el sistema de archivos FAT proporcionado por ESP-\/\+IDF.

Funcionalidades implementadas:\+
\begin{DoxyItemize}
\item Inicialización del bus SPI.
\item Montaje del sistema de archivos FAT.
\item Registro persistente de datos.
\item Almacenamiento de mensajes MQTT pendientes.
\item Recuperación de mensajes pendientes tras reconexión.
\end{DoxyItemize}\hypertarget{sd__card_8c_sd_role}{}\doxysubsection{\texorpdfstring{Rol dentro del sistema}{Rol dentro del sistema}}\label{sd__card_8c_sd_role}
Este módulo forma parte del subsistema de persistencia del sistema de fermentación de café.

Se utiliza para:\+
\begin{DoxyItemize}
\item Almacenar mediciones ambientales.
\item Registrar eventos del sistema.
\item Mantener mensajes MQTT durante pérdidas de conectividad.
\end{DoxyItemize}\hypertarget{sd__card_8c_mqtt_store_forward}{}\doxysubsection{\texorpdfstring{Estrategia Store-\/and-\/\+Forward}{Estrategia Store-\/and-\/\+Forward}}\label{sd__card_8c_mqtt_store_forward}
Cuando el broker MQTT no está disponible:\+

Sensor -\/\texorpdfstring{$>$}{>} SD

Al recuperarse la conexión:\+

SD -\/\texorpdfstring{$>$}{>} MQTT Broker

Esto garantiza que no se pierdan mediciones durante fallos temporales de red.

\begin{DoxyNote}{Nota}
El acceso concurrente debe protegerse mediante mutex.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
Escrituras simultáneas sin sincronización pueden provocar corrupción del sistema de archivos FAT.
\end{DoxyWarning}
