\doxysubsection{Descripción detallada}
Implementación del driver para sensor DHT11. 

Este módulo implementa la comunicación de bajo nivel con el sensor DHT11 utilizando el protocolo propietario basado en temporización de pulsos.

El controlador realiza:\+
\begin{DoxyItemize}
\item Generación de la señal de inicio.
\item Recepción de los 40 bits transmitidos por el sensor.
\item Verificación de integridad mediante checksum.
\item Conversión de los datos recibidos a temperatura y humedad.
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
El protocolo del DHT11 depende de retardos en microsegundos. 

Se recomienda evitar interrupciones prolongadas durante la lectura. 
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
El acceso concurrente al sensor debe protegerse mediante mutex cuando sea utilizado desde múltiples tareas Free\+RTOS.
\end{DoxyWarning}

Realiza una lectura completa del sensor DHT11. 

Ejecuta la secuencia completa de comunicación:\+
\begin{DoxyEnumerate}
\item Envía la señal de inicio al sensor.
\item Espera la respuesta de presencia.
\item Lee los 40 bits de información transmitidos.
\item Verifica la integridad mediante checksum.
\item Extrae los valores de temperatura y humedad.
\end{DoxyEnumerate}

Los datos obtenidos se almacenan en la estructura proporcionada por el usuario.
