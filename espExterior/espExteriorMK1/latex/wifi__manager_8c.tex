\doxysubsection{Descripción detallada}
Implementación del gestor de conectividad Wi\+Fi. 

Este módulo implementa la inicialización y administración de la interfaz Wi\+Fi del ESP32 utilizando ESP-\/\+IDF.

Funcionalidades:\+
\begin{DoxyItemize}
\item Inicialización de NVS.
\item Configuración del stack TCP/\+\+IP.
\item Gestión de conexión Wi\+Fi en modo estación.
\item Reintentos automáticos de conexión.
\item Obtención de dirección IP.
\item Escaneo de redes disponibles.
\end{DoxyItemize}

Aspectos de Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Event Groups para sincronización de estados.
\item Tareas bloqueantes mediante x\+Event\+Group\+Wait\+Bits().
\end{DoxyItemize}

La comunicación entre los eventos del stack Wi\+Fi y las tareas de aplicación se realiza mediante bits de estado almacenados en un Event Group.
