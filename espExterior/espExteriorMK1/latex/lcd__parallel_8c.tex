\doxysubsection{Descripción detallada}
Implementación del driver para pantalla LCD 16x2 basada en HD44780. 

Este módulo implementa la comunicación paralela en modo de 4 bits utilizando GPIO del ESP32 para controlar una pantalla LCD 16x2.

Funcionalidades implementadas:\+
\begin{DoxyItemize}
\item Inicialización del controlador HD44780.
\item Envío de comandos.
\item Envío de caracteres.
\item Posicionamiento del cursor.
\item Limpieza de pantalla.
\end{DoxyItemize}\hypertarget{lcd__parallel_8c_lcd_system_role}{}\doxysubsection{\texorpdfstring{Rol dentro del sistema}{Rol dentro del sistema}}\label{lcd__parallel_8c_lcd_system_role}
Este módulo forma parte del ESP32 externo y proporciona la interfaz visual local para el monitoreo del proceso de fermentación.

La información mostrada puede incluir:\+
\begin{DoxyItemize}
\item Temperatura ambiente.
\item Humedad relativa.
\item Estado de conexión Wi\+Fi.
\item Estado MQTT.
\item Estado del almacenamiento SD.
\item Mensajes de diagnóstico.
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
No es seguro para acceso concurrente. 

Se recomienda que únicamente la tarea task\+\_\+display acceda directamente a este módulo.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
La comunicación requiere retardos precisos para cumplir las especificaciones del controlador HD44780.
\end{DoxyWarning}
