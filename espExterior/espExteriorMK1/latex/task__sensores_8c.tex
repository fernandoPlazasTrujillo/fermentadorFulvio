\doxysubsection{Descripción detallada}
Tarea de adquisición de datos ambientales. 

Esta tarea realiza una medición única de los sensores del sistema:\+
\begin{DoxyItemize}
\item Sensor DHT11 (temperatura y humedad)
\item RTC DS3231 (fecha y hora)
\end{DoxyItemize}

Luego:\+
\begin{DoxyItemize}
\item Empaqueta los datos en una estructura
\item Envía la información a otras tareas mediante colas
\item Finaliza su ejecución
\end{DoxyItemize}

Flujo de ejecución:\+
\begin{DoxyEnumerate}
\item Lectura del sensor DHT11
\item Lectura del RTC
\item Validación de datos
\item Envío a colas:\+
\begin{DoxyItemize}
\item cola\+\_\+datos → task\+\_\+logger
\item cola\+\_\+display → task\+\_\+display
\item cola\+\_\+mqtt → task\+\_\+mqtt
\end{DoxyItemize}
\item Eliminación de la tarea
\end{DoxyEnumerate}

Comunicación:\+
\begin{DoxyItemize}
\item Salida:\+
\begin{DoxyItemize}
\item cola\+\_\+datos
\item cola\+\_\+display
\item cola\+\_\+mqtt
\end{DoxyItemize}
\end{DoxyItemize}

Aspectos de Free\+RTOS utilizados:\+
\begin{DoxyItemize}
\item Queue para distribución de datos entre tareas.
\item Ejecución one-\/shot mediante creación y destrucción dinámica.
\item Bloqueo seguro mediante port\+MAX\+\_\+\+DELAY durante el envío.
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
Esta tarea es de tipo "{}one-\/shot"{} (ejecución única).

La periodicidad de ejecución es gestionada por la tarea principal del sistema, manteniendo esta tarea enfocada exclusivamente en la adquisición de datos.
\end{DoxyNote}
