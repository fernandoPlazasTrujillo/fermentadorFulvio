\doxysubsection{Descripción detallada}
Módulo de gestión de tarjeta SD mediante interfaz SPI. 

Este módulo proporciona una capa de abstracción para el acceso a una tarjeta SD utilizando el sistema de archivos FAT y la capa VFS de ESP-\/\+IDF.

Funcionalidades principales:\+
\begin{DoxyItemize}
\item Inicialización y montaje de la tarjeta SD.
\item Registro persistente de datos de sensores.
\item Almacenamiento de eventos del sistema.
\item Buffer persistente para mensajes MQTT pendientes.
\end{DoxyItemize}\hypertarget{sd__card_8h_sd_architecture}{}\doxysubsection{\texorpdfstring{Integración en el sistema}{Integración en el sistema}}\label{sd__card_8h_sd_architecture}
Dentro del sistema de fermentación de café, este módulo cumple dos funciones fundamentales:\+


\begin{DoxyItemize}
\item Almacenamiento histórico de mediciones ambientales.
\item Persistencia temporal de mensajes MQTT cuando la conexión con el broker no está disponible.
\end{DoxyItemize}

Esta estrategia permite mantener la integridad de los datos incluso ante fallos de red o reinicios inesperados.

\begin{DoxyNote}{Nota}
Este módulo no es thread-\/safe. 

El acceso concurrente debe protegerse mediante mutex SPI.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
El acceso simultáneo sin sincronización puede provocar corrupción del sistema de archivos FAT.
\end{DoxyWarning}
