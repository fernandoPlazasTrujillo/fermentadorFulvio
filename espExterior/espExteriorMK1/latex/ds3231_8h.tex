\doxysubsection{Descripción detallada}
Interfaz pública del controlador para el RTC DS3231. 

Este módulo proporciona las funciones necesarias para interactuar con el reloj de tiempo real (RTC) DS3231 mediante comunicación I2C.

Funcionalidades principales:\+
\begin{DoxyItemize}
\item Lectura de fecha y hora.
\item Configuración de fecha y hora.
\item Configuración de alarmas periódicas.
\item Gestión de interrupciones generadas por alarmas.
\end{DoxyItemize}

\begin{DoxyNote}{Nota}
El acceso al bus I2C debe protegerse mediante mutex cuando sea compartido entre múltiples tareas Free\+RTOS.
\end{DoxyNote}
\hypertarget{ds3231_8h_rtc_architecture}{}\doxysubsection{\texorpdfstring{Arquitectura del sistema}{Arquitectura del sistema}}\label{ds3231_8h_rtc_architecture}
En el sistema de fermentación de café, el DS3231 actúa como referencia temporal principal para:\+


\begin{DoxyItemize}
\item Generación de timestamps para almacenamiento en SD.
\item Sincronización de mediciones de sensores.
\item Activación periódica mediante interrupciones.
\item Implementación de estrategias de ahorro energético.
\end{DoxyItemize}

Limpia el indicador interno de alarma. 

Limpia el indicador de alarma A1F.

Después de generarse una interrupción, el DS3231 mantiene activo el bit A1F dentro del registro de estado.

Esta función limpia dicho bit para permitir que futuras alarmas vuelvan a generar interrupciones.
